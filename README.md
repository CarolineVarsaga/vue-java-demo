# Demo Project - Vue & Java

## Tech stack

- HTML5
- Tailwind CSS
- Vue.js
- TypeScript
- Vite
- ESLint
- Java
- Spring Boot
- MySQL

## About this project

This project was created for the purpose of practicing Vue and Java together, as a headless API. The focus of this project is not on design, but rather on functionality.
